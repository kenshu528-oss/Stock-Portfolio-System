# Product Overview

Stock Portfolio System is a web-based stock portfolio management application designed for individual investors in Taiwan. The system helps users track stock holdings, price changes, dividend income, and investment performance across multiple brokerage accounts.

## Key Features

- **Multi-account portfolio management** - Separate tracking for different brokerage accounts
- **Real-time stock price integration** - Primary integration with Taiwan Stock Exchange (TWSE) API, with Yahoo Finance and Investing.com as fallbacks
- **Comprehensive dividend tracking** - Automatic dividend calculation and cost basis adjustment
- **Privacy protection** - Hide sensitive financial information with privacy mode
- **Cross-device synchronization** - Data sync via GitHub Gist integration
- **Responsive design** - Optimized for both mobile and desktop usage
- **Data import/export** - Support for JSON, CSV, and Excel formats

## Target Users

Individual investors in Taiwan who need to manage stock portfolios across multiple brokerage accounts, with emphasis on dividend tracking and cross-device accessibility.

## Language

The application interface is primarily in Traditional Chinese (繁體中文), targeting Taiwanese users and Taiwan stock market data.
