# 穩定版本記錄 (Stable Version)

**發布日期**：2026-01-23

**穩定性**：⭐⭐⭐⭐⭐ 高度穩定

### 📋 版本重點

#### 🔍 **個股搜尋功能全面優化**
- **修復核心問題**：解決個股搜尋時出現的 404 錯誤和後端連接問題
- **建立完整更新機制**：實現股票清單的自動檢測和更新流程
- **技術債務清理**：修復 Unicode 編碼、API 端點、檔案路徑等底層問題

#### 🛠️ **股票清單更新機制**
- **自動檢測**：系統啟動時自動檢查股票清單是否為最新版本
- **智能更新**：檢測到過期清單時自動在背景更新，不干擾用戶操作
- **檔案管理**：建立 7 天歷史檔案保留機制，自動清理過期檔案
- **強制更新**：支援手動強制重新下載最新股票清單

#### 🐛 **技術問題修復**
- **編碼問題**：修復 Windows cp950 編碼錯誤，解決 Unicode 字符顯示問題
- **API 端點**：修正前端與後端的 API 調用路徑 (localhost:3001)
- **檔案路徑**：統一 Python 腳本的檔案創建位置，確保前後端一致性
- **React 優化**：改善開發環境下的重複執行警告問題
