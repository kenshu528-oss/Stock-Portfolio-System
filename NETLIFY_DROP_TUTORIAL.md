# Netlify Drop 詳細教學

## 🎯 什麼是 Netlify Drop？

Netlify Drop 是一個**完全免費**的網頁託管服務，讓你可以：
- ✅ **拖拉檔案就完成部署**
- ✅ **不需要註冊帳號**
- ✅ **立即獲得網址**
- ✅ **其他人可以直接使用**

## 📋 準備工作

### 確認檔案結構
```
你的專案資料夾應該包含：
📁 Stock-Accumulation/

│   ├── 📄 styles.css
│   ├── 📄 stock-api.js

│   └── 📄 version.js

└── 📁 tests/
```

### 檢查重點
- ✅ `index.html` 必須在根目錄
- ✅ 所有檔案都在資料夾內
- ✅ 沒有缺少任何檔案

## 🚀 詳細操作步驟

### 步驟 1：開啟 Netlify Drop
```
1. 開啟瀏覽器（建議使用 Chrome 或 Edge）
2. 前往：https://app.netlify.com/drop
3. 你會看到一個大大的虛線框，寫著 "Drag and drop your site folder here"
```

### 步驟 2：準備資料夾
```
1. 開啟檔案總管
2. 找到你的專案資料夾（包含 index.html 的那個）
3. 確認資料夾名稱（建議改為簡單的英文名稱，如 "stock-system"）
```

### 步驟 3：拖拉上傳
```
1. 將整個專案資料夾拖拉到 Netlify Drop 的虛線框內
2. 看到 "Uploading..." 字樣
3. 等待上傳完成（通常 1-3 分鐘，依檔案大小而定）
```

### 步驟 4：獲得網址
```
上傳完成後，你會看到：
✅ "Your site is live at: https://amazing-name-123456.netlify.app"

這就是你的網站網址！
```

## 🎨 自訂網站名稱（可選）

### 註冊 Netlify 帳號
```
1. 前往：https://netlify.com
2. 點擊 "Sign up"
3. 可用 Google、GitHub 或 Email 註冊
```

### 自訂名稱
```
1. 登入後，找到你的網站
2. 點擊 "Site settings"
3. 點擊 "Change site name"
4. 輸入想要的名稱（如：my-stock-system）
5. 新網址：https://my-stock-system.netlify.app
```

## ⚠️ 注意事項

### 檔案大小限制
```
✅ 免費版限制：
- 單次上傳：100MB
- 每月流量：100GB
- 對一般使用綽綽有餘

❌ 如果超過限制：
- 考慮移除測試檔案
- 壓縮圖片檔案
- 或升級付費方案
```

### 檔案類型限制
```
✅ 支援的檔案：
- HTML, CSS, JavaScript
- 圖片檔案（JPG, PNG, GIF）
- 字型檔案
- JSON, XML 等資料檔案

❌ 不支援：
- 伺服器端程式（PHP, Python）
- 資料庫檔案
- 執行檔（.exe）
```

### 安全性考量
```
⚠️ 重要提醒：
- 不要上傳包含密碼的檔案
- 不要上傳個人敏感資料
- API 金鑰等機密資訊要特別注意
- 所有檔案都是公開可存取的
```

## 🛠️ 常見問題解決

### 問題 1：網站無法載入
```
可能原因：
- index.html 不在根目錄
- 檔案路徑錯誤
- 瀏覽器快取問題

解決方法：
1. 確認 index.html 在資料夾最外層
2. 檢查所有檔案路徑
3. 清除瀏覽器快取重新載入
```

### 問題 2：功能無法正常運作
```
可能原因：
- JavaScript 檔案遺失
- 檔案路徑錯誤
- 瀏覽器不支援某些功能

解決方法：
1. 按 F12 開啟開發者工具
2. 查看 Console 是否有錯誤訊息
3. 確認所有 JS 和 CSS 檔案都已上傳
```

### 問題 3：股價無法更新
```
可能原因：
- API 服務暫時無法使用
- 網路連線問題
- CORS 跨域問題

解決方法：
1. 檢查網路連線
2. 稍後再試
3. 聯絡原作者回報問題
```

### 問題 4：上傳失敗
```
可能原因：
- 檔案太大
- 網路不穩定
- 瀏覽器問題

解決方法：
1. 檢查檔案大小是否超過 100MB
2. 嘗試重新上傳
3. 換個瀏覽器試試
4. 檢查網路連線
```
