一個功能完整的股票投資組合管理系統，採用純前端技術開發。

## 🚀 特色功能

- **多帳戶管理**: 支援多個券商帳戶
- **即時股價**: 整合多個 API 來源
- **雲端同步**: 使用 GitHub Gist 跨裝置同步
- **響應式設計**: 支援手機、平板、電腦
- **離線支援**: 本地資料儲存

## 🛠️ 技術架構

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **API**: Yahoo Finance + 台灣證交所
- **儲存**: localStorage + GitHub Gist
- **部署**: GitHub Pages / Netlify / Vercel
