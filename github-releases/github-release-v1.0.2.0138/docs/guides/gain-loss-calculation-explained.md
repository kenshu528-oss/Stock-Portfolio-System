# 總損益計算說明 (Gain/Loss Calculation Explained)

## 📊 總損益到底怎麼算的？

### 基本概念

**總損益 = 如果現在全部賣掉，能賺（或賠）多少錢**

但要考慮：
1. **買入成本**（包含買入手續費）
2. **賣出收入**（扣除賣出手續費和證交稅）
3. **除權息影響**（根據模式決定）

---

## 🔢 計算公式

### 單支股票的損益

```
損益 = 淨賣出收入 - 總買入成本
```

#### 1. 總買入成本
```
總買入成本 = (持股數 × 成本價) + 買入手續費

買入手續費 = MAX(20元, 買入金額 × 0.1425%)
```

#### 2. 淨賣出收入
```
淨賣出收入 = (持股數 × 現價) - 賣出手續費 - 證交稅

賣出手續費 = MAX(20元, 賣出金額 × 0.1425%)
證交稅 = 賣出金額 × 稅率
  - 一般股票：0.3%
  - 債券ETF：0.1%
```

---

## 🎯 兩種損益模式

### 模式 1：原始損益（excluding_rights）

**只看股價漲跌，不考慮除權息**

使用**原始成本價**計算：

```
範例：2886 兆豐金
- 購買：1000股 × $33.00 = $33,000
- 買入手續費：$33,000 × 0.1425% = $47（最低20元，實際47元）
- 總買入成本：$33,000 + $47 = $33,047

- 現價：$40.50
- 賣出金額：1000股 × $40.50 = $40,500
- 賣出手續費：$40,500 × 0.1425% = $58
- 證交稅：$40,500 × 0.3% = $122
- 淨賣出收入：$40,500 - $58 - $122 = $40,320

損益 = $40,320 - $33,047 = $7,273
```

**特點**：
- ✅ 簡單直觀，只看股價變化
- ❌ 忽略了配股和股息的影響
- 適合：想知道純粹股價漲跌的獲利

---

### 模式 2：含除權息（including_rights）

**包含股息和配股的完整投資報酬**

使用**調整後成本價**計算：

```
範例：2886 兆豐金（有配股和股息）
- 原始購買：1000股 × $33.00 = $33,000
- 2024除權息：現金股利 $1.5/股，配股 30‰
  - 收到現金：1000股 × $1.5 = $1,500
  - 配股：1000股 × 30‰ = 30股
  - 除權後持股：1030股
  - 調整後成本價：($33,000 - $1,500) ÷ 1030 = $30.58

- 2025除權息：現金股利 $1.6/股，無配股
  - 收到現金：1030股 × $1.6 = $1,648
  - 持股不變：1030股
  - 調整後成本價：($30.58 × 1030 - $1,648) ÷ 1030 = $28.98

- 買入手續費：$33,000 × 0.1425% = $47
- 總買入成本：(1030股 × $28.98) + $47 = $29,896

- 現價：$40.50
- 賣出金額：1030股 × $40.50 = $41,715
- 賣出手續費：$41,715 × 0.1425% = $59
- 證交稅：$41,715 × 0.3% = $125
- 淨賣出收入：$41,715 - $59 - $125 = $41,531

損益 = $41,531 - $29,896 = $11,635
```

**特點**：
- ✅ 反映完整的投資報酬
- ✅ 考慮了配股增加的持股數
- ✅ 考慮了股息降低的成本價
- 適合：想知道真實的投資獲利

## 📈 總損益計算

**總損益 = 所有股票的損益加總**

```
總損益 = 股票1損益 + 股票2損益 + 股票3損益 + ...
```

## 🔄 兩種模式的差異

### 原始損益模式（excluding_rights）
- 使用原始成本價
- 不考慮配股和股息
- 損益會比較**保守**（因為沒算股息收入）

### 含除權息模式（including_rights）
- 使用調整後成本價
- 考慮配股和股息
- 損益會比較**真實**（反映完整報酬）

### 實際差異範例

```
2886 兆豐金：
- 原始損益：約 $7,273（只看股價）
- 含除權息：約 $11,635（包含配股和股息）
- 差異：$4,362（這是配股和股息的貢獻）
```

## 📊 交易成本影響

### 為什麼要考慮交易成本？

**因為這是真實的獲利！**

```
假設不考慮交易成本：
- 買入：1000股 × $33 = $33,000
- 賣出：1000股 × $40.5 = $40,500
- 看起來賺：$7,500

實際考慮交易成本：
- 買入成本：$33,000 + $47（手續費）= $33,047
- 賣出收入：$40,500 - $58（手續費）- $122（證交稅）= $40,320
- 實際賺：$7,273

差異：$227（約3%的獲利被交易成本吃掉）
```

### 交易成本組成

1. **買入手續費**：0.1425%（最低20元）
2. **賣出手續費**：0.1425%（最低20元）
3. **證交稅**：
   - 一般股票：0.3%
   - 債券ETF：0.1%

**總計約 0.585%**（一般股票）

## 🎯 總結

### 總損益計算流程

```
1. 對每支股票：
   ├─ 計算總買入成本（含手續費）
   ├─ 計算淨賣出收入（扣手續費和證交稅）
   └─ 損益 = 淨賣出收入 - 總買入成本

2. 加總所有股票的損益 = 總損益

3. 根據模式選擇成本價：
   ├─ 原始損益：使用原始成本價
   └─ 含除權息：使用調整後成本價
```

主要原因：
