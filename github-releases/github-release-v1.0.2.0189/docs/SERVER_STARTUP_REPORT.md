## 🐛 常見問題

### Q1: 無法訪問 http://localhost:5173？
**A**: 檢查防火牆設定，或嘗試使用 `http://127.0.0.1:5173`

### Q2: 看不到改善效果？
**A**: 
1. 清除瀏覽器快取（Ctrl+Shift+Delete）
2. 硬重載（Ctrl+Shift+R）
3. 確認版本號是 v1.0.2.0166

### Q3: 後端 API 無法連接？
**A**: 
1. 檢查後端是否在 port 3001 運行
2. 查看後端輸出：`getProcessOutput --processId 26`
3. 測試健康檢查：訪問 `http://localhost:3001/health`

### Q4: 數值沒有千分位逗號？
**A**: 
1. 確認版本號是 v1.0.2.0166
2. 檢查數值是否大於 1000（小於 1000 不會顯示逗號）
3. 清除瀏覽器快取
