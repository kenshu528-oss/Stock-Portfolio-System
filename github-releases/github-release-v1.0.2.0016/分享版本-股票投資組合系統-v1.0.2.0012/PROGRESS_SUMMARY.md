## 📊 資料結構

### 核心資料模型
- **Account**: 投資帳戶
- **Stock**: 股票持倉
- **DividendRecord**: 股息記錄
- **OperationLog**: 操作日誌

### 匯出格式
```json
{
  "version": "1.0.1.0038",
  "exportDate": "2025-01-11T...",
  "accounts": [...],
  "stocks": [...],
  "metadata": {...}
}
```
