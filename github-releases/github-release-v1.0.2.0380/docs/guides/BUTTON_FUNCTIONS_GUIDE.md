# 📊 Header 按鈕功能說明

## 🔄 損益模式切換按鈕（第一個）

### 功能
簡單的開關按鈕，控制是否在損益計算中包含股息收入。

### 兩種模式
- **不含股息**：純粹的股價損益 = (現價 - 成本價) × 股數
- **含股息**：股價損益 + 股息收入

### 圖示說明
- 📈 純趨勢線：不含股息
- 📈$ 趨勢線+$符號：含股息

---

## 🎯 除權息模式切換按鈕（第二個）

### 功能
循環切換四種精細的除權息計算模式，處理配股配息對投資績效的影響。

### 四種模式詳解

#### 1️⃣ 原始損益（不含除權息）
- **計算**：完全使用原始買入價格
- **適用**：想看純粹的股價變動
- **圖示**：📊原 - 柱狀圖+「原」字

#### 2️⃣ 含現金股利
- **計算**：原始損益 + 現金股息
- **適用**：只考慮現金收入，不考慮配股
- **圖示**：💰現金 - 圓圈+十字+「現金」

#### 3️⃣ 完整除權息（推薦）
- **計算**：考慮配股對成本價的稀釋效應
- **適用**：最完整的投資績效分析
- **圖示**：📊💰除權息 - 股票分割+錢幣+「除權息」

#### 4️⃣ 調整成本損益
- **計算**：使用除權息調整後的成本價
- **適用**：與券商APP計算方式一致
- **圖示**：🧮調整 - 計算機+「調整」

---

## 💡 使用建議

### 一般投資人
- 使用「完整除權息」模式，最能反映真實投資績效

### 短線交易者
- 使用「原始損益」模式，專注於股價波動

### 存股族
- 使用「含現金股利」或「完整除權息」模式，重視股息收入

### 與券商對帳
- 使用「調整成本損益」模式，與券商APP計算一致

---

## 🔍 實際範例

假設您買入台積電1000股，成本價500元：

### 情況：股價漲到550元，配息10元，配股100股

| 模式 | 計算方式 | 結果 |
|------|----------|------|
| 原始損益 | (550-500)×1000 | +50,000元 |
| 含現金股利 | 50,000 + 10×1000 | +60,000元 |
| 完整除權息 | 考慮配股稀釋後計算 | +55,000元 |
| 調整成本損益 | 使用調整成本價計算 | +52,000元 |

## ❓ 常見問題

**Q: 兩個按鈕有什麼差別？**
A: 第一個是簡單開關（含息/不含息），第二個是精細模式（四種計算方式）

**Q: 應該用哪個模式？**
A: 建議使用「完整除權息」模式，最能反映真實投資績效

**Q: 為什麼數字會不一樣？**
A: 不同模式考慮的因素不同，配股會影響成本價，配息會增加收入

**Q: 哪個模式最準確？**
A: 「完整除權息」模式最準確，完整考慮了除權息對投資的影響
