========================================
股票投資組合系統 v1.0.2.0012 - 使用說明
========================================

🚀 快速開始（3步驟）：

1. 【安裝】雙擊「安裝依賴.bat」（首次使用必須執行）
2. 【啟動】雙擊「啟動系統.bat」
3. 【使用】瀏覽器會自動開啟到 http://localhost:5173

📋 系統需求：
- Windows 10/11
- Node.js 16+ （請先到 https://nodejs.org/ 下載安裝）
- 網路連線

🛠️ 其他啟動方式：
- 「啟動前端.bat」- 只啟動前端服務
- 「啟動後端.bat」- 只啟動後端服務
- 「停止系統.bat」- 停止所有服務

⚠️ 注意事項：
- 首次使用必須先執行「安裝依賴.bat」
- 啟動後請保持命令視窗開啟
- 關閉瀏覽器不會停止服務，需要關閉命令視窗

📊 主要功能：
✅ 股票查詢與價格監控
✅ 投資組合管理
✅ 資料匯入/匯出
✅ 雲端同步
✅ 響應式設計

🔧 故障排除：
- 如果無法啟動：檢查是否已安裝 Node.js
- 如果端口被占用：重新啟動電腦後再試
- 如果股票查詢失敗：檢查網路連線
