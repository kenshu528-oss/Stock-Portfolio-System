## 方法 1：Netlify Drop（推薦⭐⭐⭐⭐⭐）

### 為什麼推薦？
- ✅ **完全免費**
- ✅ **不需註冊帳號**
- ✅ **拖拉就完成**
- ✅ **自動獲得網址**
- ✅ **其他人直接用瀏覽器開啟**

### 操作步驟
```
1. 前往網站：https://app.netlify.com/drop

2. 選取你的專案資料夾
   （包含 index.html 和所有檔案的資料夾）

3. 直接拖拉到網頁上的虛線框內

4. 等待上傳（通常 1-2 分鐘）

5. 完成！獲得網址（例如：https://wonderful-curie-123456.netlify.app）

6. 複製網址分享給朋友
```

### 分享方式
```
📱 LINE/WhatsApp：直接貼上網址
📧 Email：在信件中包含網址
📋 便條紙：寫下網址給朋友
```

## 方法 2：Google Drive 分享

### 為什麼選擇？
- ✅ **大家都有 Google 帳號**
- ✅ **操作簡單熟悉**
- ✅ **可以下載使用**
- ✅ **不會過期**

### 操作步驟
```
1. 將專案資料夾壓縮成 ZIP 檔案
   （右鍵點擊資料夾 → 傳送到 → 壓縮的資料夾）

2. 上傳到 Google Drive

3. 右鍵點擊 ZIP 檔案 → "取得連結"

4. 設定為 "知道連結的使用者"

5. 複製連結分享
```

### 使用說明（給朋友）
```
📥 下載步驟：
1. 點擊分享的連結
2. 點擊 "下載" 按鈕
3. 解壓縮檔案
4. 雙擊 "index.html" 開始使用

💡 提醒：
- 資料會儲存在瀏覽器中
- 建議定期備份資料
- 可以在任何電腦使用
```

## 方法 3：USB 隨身碟分享

### 為什麼選擇？
- ✅ **最直接的方式**
- ✅ **面對面分享**
- ✅ **不需網路**
- ✅ **100% 隱私**

### 操作步驟
```
1. 將整個專案資料夾複製到 USB

2. 建立使用說明檔案：
   新增文字檔 "使用說明.txt"

3. 在說明檔中寫入：
   「雙擊 index.html 檔案即可開始使用存股紀錄系統」

4. 將 USB 給朋友

5. 朋友將檔案複製到電腦後即可使用
```

## 🚀 進階技巧

```
1. 在 Netlify 註冊免費帳號
2. 可以自訂網址名稱

```

### 定期更新
```
1. 關注原作者的更新通知
2. 重新上傳新版本到相同位置
3. 網址保持不變，功能自動更新
```

```
1. 使用 QR Code 產生器（如：qr-code-generator.com）
2. 輸入你的分享網址
3. 下載 QR Code 圖片
4. 可以印製或分享圖片，掃描即可使用
```
