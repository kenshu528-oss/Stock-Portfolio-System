# 雲端同步快速設定（5分鐘完成）

## 🚀 快速設定流程

### 步驟 1：註冊 GitHub（2分鐘）
```
1. 前往：https://github.com
2. 點擊「Sign up」
3. 填入 Email、用戶名、密碼
4. 完成 email 驗證
```

### 步驟 2：建立 Token（2分鐘）
```
1. 登入 GitHub
2. 點擊右上角頭像 → Settings
3. 左側選單：Developer settings
4. Personal access tokens → Tokens (classic)
5. Generate new token (classic)
6. 填入：
   - Note: 存股系統
   - Expiration: 90 days
   - 勾選：✅ gist
7. Generate token
8. 複製 Token（重要！）
```

### 步驟 3：在存股系統設定（1分鐘）
```
1. 點擊存股系統右上角「☁️」
2. 貼上 Token
3. Gist ID 留空
4. 點擊「儲存設定」
5. 點擊「立即同步」
```

## ✅ 完成！

現在你的資料會自動備份到雲端，可以在任何裝置使用！

---

## 📱 其他裝置設定

### 在新裝置使用
```
1. 開啟存股系統
2. 點擊「☁️」
3. 輸入相同的 Token 和 Gist ID
4. 點擊「從雲端下載」
5. 完成！
```

---

## 🔧 如果遇到問題

### Token 過期
```
重新建立 Token，更新設定即可
```

### 同步失敗
```
檢查網路連線和 Token 是否正確
```

### 找不到資料
```
確認 Gist ID 是否正確
```
